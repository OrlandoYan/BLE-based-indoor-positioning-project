The Profiles_LowPower and the Profiles_DMA_LowPower application projects 
come with an App for Android devices: STM32_BLE_Profiles.apk.

The STM32_BLE_Profiles App can be installed on any Android smartphone equipped with
a Bluetooth Low Energy (BLE) chip and Android OS >= ver 4.3 or newer.
For a more detailed description of the App, please refer to the link on Google Play:
https://play.google.com/store/apps/details?id=com.stm.bluetoothlevalidation

The STM32_BLE_Profiles app is also available on Apple Store (iTunes) at https://itunes.apple.com/it/app/stm32-ble-toolbox/id1081331769?mt=8 

The Profiles_LowPower application project can be found in this package at:
$BASE_DIR\Projects\Multi\Applications\Profiles_LowPower
The Profiles_DMA_LowPower application project can be found in this package at:
$BASE_DIR\Projects\Multi\Applications\Profiles_DMA_LowPower
(for instructions on how to install and use them, please, refer to readme.txt at
the root directory of this package).

The STM32_BLE_Profiles App, once installed, is able to show in readable form and with
a nice and simple graphical user interface all notifications coming from the STM32
board where the Profiles_LowPower (or Profiles_DMA_LowPower) firmware is running.
